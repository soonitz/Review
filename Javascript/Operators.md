# 산술 연산자(Arithmetic Operator)
<br>

| Command | Description |
|:---:|:---|
| ++ | increment |
| -- | decrement |
| == | equal to |
| === | exactly equal to |
| != | not equal |
| !== | not equal (different value or different type) |
